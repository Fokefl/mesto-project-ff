# Mesto

Проектная работа — интерактивная страница, на которой пользователи могут делиться фотографиями мест.

## Функциональность
- Загрузка карточек и данных пользователя с сервера
- Редактирование профиля и аватара
- Добавление и удаление карточек
- Лайки карточек
- Валидация всех форм
